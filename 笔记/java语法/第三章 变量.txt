1、字面量

   1.1什么是字面量？
      运用java语言编程去解决现实中的问题时，会面临许多的处理数据
      在java语言中“数据”被称为“字面量”。
      10
      1.23
      true
      false
      'a'
      "abc"
      以上这些都是数据，在程序中被叫做“字面量”。
      
      字面量可以分为很多种类：
      浮点型字面量：1.1 1.333....
      整型字面量：1 2 3 4...
      布尔型字面量：true false
      字符型字面量：'a','b','c'...
      字符串字面量：“奥德赛大所多撒”,"asdhhasgd".....

      其中字符型和字符串型都是描述了现实中的文字
      需要注意的是 "" 与 ''的区别。

    1.2、变量
	在内存存储数据的最基本的单元。

	数据类型有什么用？
		不同的数据类型，在内存分配的空间大小不同。
		也就是说，Java虚拟机到低给这个数据分配多大的空间，
		主要还是看这个变量的数据类型。
		根据不同的类型，分配不同大小的空间。
		对于int这种整数类型，jvm会自动给int分配4个字节大小的空间。
	
	变量三要素：
		数据类型、变量名、变量的值。

	在java语言中有一个规定，变量声明后 ，必须先赋值在访问。
	
	在同一个域当中，变量名不能重复声明，因为不可能有两个内存空间是同一个名字。

	不同的数据类型也不能重复命名。
	
	int a,b,c=100;   a、b没有赋值，c赋值了

	关于变量的一个分类
		变量根据出现的位置进行划分
		 在方法体中声明的变量：局部变量
			局部变量的生命周期 是从方法开始执行到方法执行结束

		 在方法体外，类体内声明的变量：成员变量

	  注意：局部变量只在方法体当中有效，方法体执行结束该变量的内存就释放了
